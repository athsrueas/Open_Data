Bulk Data Download

The archives listed below contain the latest official data disseminated by the UNESCO Institute for Statistics (UIS). This archive is a result of a rigorous data production activity that ensures a high level of data quality. In order to expose the dataset to the largest audience possible and reduce its complexity, the UIS has normalized and compiled it using comma-separated values (CSV) format. This allows both beginners and seasoned data professionals to understand and work with it.

NOTE: These files are intended for programmatic processing. Some desktop tools (for example, Microsoft Excel) may not load or display the full dataset; to access all records, use scripting or data analysis tools such as Python or R (or preprocess the files as needed).
